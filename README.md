# ROS Image Analysis

This repository contains ROS code to analyze pictures taken from a connected HD camera. Having the [camera_2d] package is required to launch this node for the moment.

For the moment, the pictures are petri dish with colonies of bacterias.

Launch the nodes with:
  - `roslaunch ros_image_analysis image_analysis.launch`


[camera_2d]: https://github.com/biobotus/camera_2d
